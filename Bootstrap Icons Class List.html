How to use Bootstrap icons?

Step 1: Include the following Bootstrap CDN link in the <head> section of you HTML document.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.5.0/font/bootstrap-icons.css">

Step 2: To place icons into your web page use the syntax <i class="bi-*"></i> where * represent the class of a particular icon. So, if an icon class is alarm then the icon HTML code would be <i class="bi-alarm"></i>, similarly if the icon class is arrow-down the resulting icon HTML code would be <i class="bi-arrow-down"></i>, and so on.
